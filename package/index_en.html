Introduction

MUCOM88 Windows is a system to run MUCOM88 FM synthesis sound driver on NEC's PC-8801 series.
Both the command line version and the GUI version are prepared. 

		mucom88win.exe         Windows GUI editor
		mucom88.exe            Windows CLI version

By using MUCOM88 Windows, music written as MML (Music Macro Language) specialized for music description can be played in the same way as the sound source of PC-8801. In addition, performance via the actual FM sound source chip (YM2608) is supported via SCCI (Sound Chip Common Interface).

Detailed explanation about MUCOM88 Windows and MML reference are provided in Open MUCOM88 WiKi.
Please see the following URL on your web browser. 

	https://github.com/onitama/mucom88/wiki/About-MUCOM88-Windows

The latest information on MUCOM88 Windows and file downloads are available at the following site.
